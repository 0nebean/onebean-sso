热调试

如果你需要在项目中进行热调试，首先你需要将dao层所依赖的core源码包检出到Kepler同一工作空间下。

net.onebean.core包的github地址

net.onebean.core引用了 spring boot 热调试工具 ：

        <dependency>
            <groupId>org.springframework.boot</groupId>
            <artifactId>spring-boot-devtools</artifactId>
            <scope>runtime</scope>
        </dependency>

该包会检查项目中classPath中的文件变动，如果法发现文件变动会自动重启项目。

值得一说的是该工具支持LiveReload，去下载 LiveReload 的浏览器插件，浏览器会和后台建立一个websocket，检测到重启后，会自动刷新浏览器。

20.3 LiveReload
The spring-boot-devtools module includes an embedded LiveReload server that can be used to trigger a browser refresh when a resource is changed. LiveReload browser extensions are freely available for Chrome, Firefox and Safari from livereload.com.

If you do not want to start the LiveReload server when your application runs, you can set the spring.devtools.livereload.enabled property to false.
