路由与面包屑

Kepler中路由的API非常简单，你只需要在页面中以下面这行代码进行页面跳转即可：

routingPage('/dic/preview','字典管理');

入参分别是需要跳转的url地址和面包屑标题，当然面包屑的显示需要在页面中引入面包屑控件：

<th:block th:include="/public/breadCrumbs :: breadCrumbsTempl"/>
