Kepler - 简单 快速 主流 安全

Kepler是一个完全免费的j2ee开源框架

如何开始

松耦合,多模块,多端程序最大程度共用代码

主流技术 简单上手 低门槛

框架后端基于spring framework 4, spring boot 1.5,主流的渐进式框架,orm基于mybatis和自定义的sqlSessionFactory实现了默认的增删改查方法,效率高,简单易上手。

组件丰富，模块化

框架前端基于AmazeUI+art-template丰富的生态库可拓展,后端基于松耦合的多模块设计,现有众多可插拔组件如:redis、mongoDB、aliyun-oss、aliyun-sms等,并且组件库生态完全开源。

精细的权限验证

项目权限控制基于spring security实现,健壮性不言而喻,前后台双重验证,密码经过SHA256+秘钥+行内随机盐加密储存,精细的权限验证从根本上避免常见注入攻击。

一键生成代码菜单及权限

框架内置生成代码工具,提供包含增删改查的列表,树结构,一对多结构等功能的一键代码生成,并自动生成菜单和权限,如果对外观要求不是很高，生成的功能就可以用了。

丰富的组件库生态

Kepler本身是一个松耦合多模块的j2ee的快速开发框架,并不直接适用与大型分布式应用,作者开发其初心是想为自己开发中小型企业应用提供一把利器,并预留出可拓展组件模式供大家后续一起为框架添柴加火。
